Project Description

In this project, I implemented face morphing techniques to create a morph animation, compute population mean faces, and generate caricatures. This involves defining correspondences between facial keypoints, creating a morph sequence, and extrapolating from the population mean face.

Part 1: Defining Correspondences

Here, I manually selected facial keypoints on both images A and B to define correspondences between the two faces. I used Delaunay triangulation to create a mesh for morphing.

Part 2: Computing the "Mid-way Face"

Using the defined correspondences, I computed the mid-way face by averaging the shape of both images and warping them into the average shape, then blending the colors.

Bells and Whistles - PCA

I computed several PCA bases for the face space, using different numbers of principal components. I then attempted to reconstruct three of the original images using these bases to see how many principal components are necessary for a good enough reconstruction.
